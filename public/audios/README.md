# Audio-Dateien für die Agora-Audioführung\n\nDieser Ordner sollte die folgenden Audio-Dateien enthalten:\n\n## Stationen-Audio (MP3-Format empfohlen)\n\n- `station-01.mp3` - Eingang zur Agora\n- `station-02.mp3` - Stoa Poikile (Bunte Halle)\n- `station-03.mp3` - Altar der Zwölf Götter\n- `station-04.mp3` - Odeion des Agrippa\n- `station-05.mp3` - Tempel des Hephaistos\n- `station-06.mp3` - Bouleuterion (Ratshaus)\n- `station-07.mp3` - Tholos (Rundgebäude)\n- `station-08.mp3` - Metroon (Staatsarchiv)\n- `station-09.mp3` - Monument der Eponymen Heroen\n- `station-10.mp3` - Mittlere Stoa\n- `station-11.mp3` - Süd-Stoa I\n- `station-12.mp3` - Südost-Brunnenhaus\n- `station-13.mp3` - Bibliothek des Pantainos\n- `station-14.mp3` - Stoa des Attalos\n- `station-15.mp3` - Kirche der Heiligen Apostel\n\n## Audio-Spezifikationen\n\n- **Format**: MP3 oder OGG\n- **Dauer**: 3-5 Minuten pro Station\n- **Qualität**: 128 kbps oder höher\n- **Sprache**: Deutsch\n- **Inhalt**: Historische Informationen, Beschreibungen und Kontext zu den jeweiligen Stationen\n\n## Hinweise\n\n- Die Audio-Player auf der Website unterstützen sowohl MP3 als auch OGG-Formate\n- Fallback-Unterstützung ist implementiert\n- Dateien sollten aussagekräftige Namen haben, die der Station entsprechen\n- Für optimale Performance sollten Dateien komprimiert, aber von guter Qualität sein\n\n## Schulprojekt\n\nDiese Audioführung ist ein Bildungsprojekt über die antike Agora von Athen.
